Principes directeurs

P.01 Mobile-first, toujours Le viewport de référence est 375×812. Toute densité, tout hit-target (44px min), toute hiérarchie est conçue pour le pouce avant d'être enrichie sur desktop.
P.02 Parchemin clair, encre sombre Deux thèmes à parité. Le mode clair évoque le parchemin chaud ; le mode sombre évoque le cuir d'un vieux grimoire. Jamais de blanc pur, jamais de noir pur.
P.03 Ornement, jamais ornemental L'ambiance fantasy est portée par la typo et la palette, pas par des SVG décoratifs. Les composants restent géométriques, modernes, lisibles.
P.04 Tokens d'abord, classes ensuite Tout commence par une CSS custom property. Les classes utilitaires et variables Bootstrap se mappent dessus — jamais l'inverse.
