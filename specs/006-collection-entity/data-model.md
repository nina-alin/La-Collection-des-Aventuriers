# Data Model: Entité Collection et Vue Détail

**Feature**: `006-collection-entity` | **Date**: 2026-05-25

## New Entities

### Collection

| Field | PHP Type | DB Column | Nullable | Constraints |
|-------|----------|-----------|----------|-------------|
| `id` | `Uuid` | `id UUID PK` | No | Auto-generated (`Uuid::v4()`) |
| `nom` | `string` | `nom VARCHAR(255)` | No | NotBlank, Length(max:255), UniqueEntity |
| `nomOriginal` | `?string` | `nom_original VARCHAR(255)` | Yes | Length(max:255) |
| `slug` | `string` | `slug VARCHAR(255)` | No | NotBlank, Length(max:255), UniqueEntity; auto-generated by `CollectionSlugger` |
| `description` | `string` | `description TEXT` | No | NotBlank |
| `genre` | `GenreCollection` | `genre VARCHAR(50)` | No | Choice(enum cases) |
| `createurs` | `array` | `createurs JSON` | No | Default `[]`; no cardinality constraint |
| `anneeCreation` | `?int` | `annee_creation SMALLINT` | Yes | Positive, ≤ current year (if set) |
| `editeurHistorique` | `?string` | `editeur_historique VARCHAR(255)` | Yes | Length(max:255) |
| `statut` | `StatutCollection` | `statut VARCHAR(20)` | No | Choice(enum cases) |
| `imageLogo` | `?string` | `image_logo VARCHAR(255)` | Yes | Length(max:255) |

**DB Indexes**:
- `idx_collection_slug` on `slug` (URL lookup)
- `idx_collection_nom` on `nom` (uniqueness enforcement)

**Relationship**: `OneToMany → Book` (inverse side; `mappedBy: 'collection'`)

## Modified Entities

### Book (existing)

**Added field**:

| Field | PHP Type | DB Column | Nullable | Constraint |
|-------|----------|-----------|----------|------------|
| `collection` | `?Collection` | `collection_id UUID FK` | Yes | `ON DELETE SET NULL`; index `idx_book_collection_id` |

**Relationship**: `ManyToOne → Collection` (`nullable: true`, `inversedBy: 'books'`)

**Note**: The existing `saga` (string) field on `Book` is **not removed** — it holds free-text saga info that may differ from the `Collection.nom`. The two coexist; `FR-006` only requires that the `collection` relation be used for navigation links and breadcrumb.

## Database Schema (PostgreSQL)

### New table: `collection`

```sql
CREATE TABLE collection (
    id          UUID        NOT NULL PRIMARY KEY,
    nom         VARCHAR(255) NOT NULL,
    nom_original VARCHAR(255) DEFAULT NULL,
    slug        VARCHAR(255) NOT NULL,
    description TEXT         NOT NULL,
    genre       VARCHAR(50)  NOT NULL,
    createurs   JSON         NOT NULL DEFAULT '[]',
    annee_creation SMALLINT  DEFAULT NULL,
    editeur_historique VARCHAR(255) DEFAULT NULL,
    statut      VARCHAR(20)  NOT NULL,
    image_logo  VARCHAR(255) DEFAULT NULL
);

CREATE UNIQUE INDEX uniq_collection_slug ON collection (slug);
CREATE UNIQUE INDEX uniq_collection_nom  ON collection (nom);
CREATE INDEX idx_collection_slug ON collection (slug);
```

### Modified table: `book`

```sql
ALTER TABLE book
    ADD COLUMN collection_id UUID DEFAULT NULL,
    ADD CONSTRAINT fk_book_collection
        FOREIGN KEY (collection_id) REFERENCES collection (id) ON DELETE SET NULL;

CREATE INDEX idx_book_collection_id ON book (collection_id);
```

### Rollback (down)

```sql
ALTER TABLE book DROP CONSTRAINT fk_book_collection;
DROP INDEX idx_book_collection_id;
ALTER TABLE book DROP COLUMN collection_id;
DROP TABLE collection;
```
